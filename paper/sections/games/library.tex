\section{Game Library}
\label{sec:game-library}

The Kant framework provides a library of 87+ games spanning 9 strategic domains.
Each game is implemented as a \texttt{GameConfig} with formally specified payoff
structures, action spaces, and metadata tags. This section catalogues every game
by domain; Appendix~\ref{app:catalog} provides the full specification table.

\subsection{Classical Dilemmas}
\label{sec:classical}

The foundational two-player simultaneous-move games that define the core tensions
in strategic interaction.

\begin{itemize}[nosep]
    \item \textbf{Prisoner's Dilemma} --- cooperation vs.\ self-interest
    \item \textbf{Stag Hunt} --- payoff-dominant vs.\ risk-dominant equilibrium
    \item \textbf{Hawk-Dove (Chicken)} --- conflict escalation and anti-coordination
    \item \textbf{Matching Pennies} --- zero-sum mixed-strategy reasoning
\end{itemize}
